# Snoopy Holding a Heart (Turtle Graphics)

This repository contains a simple Python script that uses the `turtle` module to draw a stylized illustration of **Snoopy holding a heart**.

## Files

- `Flor Rosada.py` – The main Python script that creates the drawing.
- `README.md` – This file.

## Prerequisites

- Python 3.x (the script was tested with Python 3.11).
- The standard `turtle` library (included with the default Python installation).

## Running the script

```bash
python "Flor Rosada.py"
```

Executing the above command will open a Turtle graphics window and render the Snoopy‑style illustration.

## Customisation

Feel free to tweak the drawing by modifying the angles, radii, and colours in `Flor Rosada.py`. The `go`, `ojos`, and `circulos` helper functions make it easy to adjust positions and shapes.
